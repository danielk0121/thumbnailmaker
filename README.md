# 🖼️ Thumbnail Maker (썸네일 메이커)

HTML5 SVG와 Canvas를 활용하여 빠르고 쉽게 텍스트 기반 썸네일 이미지를 생성할 수 있는 웹 서비스입니다.

## 🚀 프로젝트 개요
단순한 텍스트 썸네일 이미지를 즉석에서 디자인하고, 다양한 포맷(PNG, JPG, SVG)으로 추출하여 사용할 수 있는 도구입니다. 복잡한 그래픽 툴 없이 브라우저에서 바로 고해상도 썸네일을 제작할 수 있습니다.

## ✨ 주요 기능

### 🎨 배경 설정 (Background)
- **단색 지정**: 원하는 배경색을 자유롭게 선택
- **랜덤 색상**: 클릭 한 번으로 무작위 색상 조합 생성
- **그라데이션**: 두 개 이상의 색상을 조합한 세련된 그라데이션 효과 지원

### ✍️ 텍스트 편집 (Text)
- **자유로운 입력**: 썸네일에 들어갈 문구 실시간 입력 및 반영
- **크기 조절**: 슬라이더를 통해 텍스트 크기 확대 및 축소
- **정렬**: 텍스트 위치 및 정렬 옵션 제공 (중앙 정렬 등)

### 📐 레이아웃 및 디자인 (Layout & Design)
- **커스텀 해상도**: 가로(Width)와 세로(Height) 크기를 직접 지정 (예: 1280x720, 1080x1080 등)
- **모서리 라운드**: 모서리를 둥글게 처리하는 둥글기(Border Radius) 조절 기능

### 💾 이미지 추출 (Export)
- **비트맵 저장**: PNG, JPG 포맷으로 고해상도 다운로드
- **벡터 저장**: 해상도 손실 없는 SVG 포맷 추출 기능 제공

## 🛠️ 기술 스택 (Planned)
- **Frontend**: React (TypeScript)
- **Styling**: Vanilla CSS
- **Graphics**: HTML5 Canvas API, SVG
- **Build Tool**: Vite

## 🏃 시작하기 (Development)

```bash
# 의존성 설치
npm install

# 로컬 서버 실행
npm run dev

# 빌드
npm run build
```
