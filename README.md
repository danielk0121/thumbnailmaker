# Thumbnail Maker (썸네일 메이커)

순수 HTML5 SVG와 Vanilla JS/CSS를 활용하여 빠르고 쉽게 텍스트 기반 썸네일 이미지를 생성할 수 있는 웹 서비스입니다.

## 프로젝트 개요
단순한 텍스트 썸네일 이미지를 디자인하고 PNG, JPG, SVG 등 다양한 포맷으로 추출할 수 있는 도구입니다.

## 폴더 구조
```text
thumbnailmaker/
├── app/                # 애플리케이션 소스 코드 (HTML, CSS, JS)
├── spec/               # 프로젝트 요구사항 및 스펙 문서 (PRD.md)
├── .gitignore          # Git 제외 목록
├── GEMINI.md           # 프로젝트 개발 규칙
└── README.md           # 프로젝트 소개
```
