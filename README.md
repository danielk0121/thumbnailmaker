# 🖼️ Thumbnail Maker (썸네일 메이커)

HTML5 SVG와 Canvas를 활용하여 빠르고 쉽게 텍스트 기반 썸네일 이미지를 생성할 수 있는 웹 서비스입니다.

## 🚀 프로젝트 개요
단순한 텍스트 썸네일 이미지를 디자인하고 PNG, JPG, SVG 등 다양한 포맷으로 추출할 수 있는 도구입니다.

## 📂 폴더 구조
```text
thumbnailmaker/
├── spec/               # 프로젝트 요구사항 및 스펙 문서 (PRD.md)
├── public/             # 정적 자산 (이미지, 파비콘 등)
├── src/                # 소스 코드
│   ├── components/     # UI 컴포넌트
│   ├── assets/         # 스타일 및 에셋
│   └── App.tsx         # 메인 애플리케이션 로직
├── .gitignore          # Git 제외 목록
├── GEMINI.md           # 프로젝트 개발 규칙
└── README.md           # 프로젝트 소개
```

## 📝 상세 스펙
- 모든 기술적 상세 사항과 요구사항은 [spec/PRD.md](./spec/PRD.md) 파일을 참조하십시오.
